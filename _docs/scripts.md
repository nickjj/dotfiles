All scripts listed below get installed by default. They get installed on all
supported systems, both on the command line and for the GUI. You can
configure `SCRIPTS_INSTALL` to customize this list.

These are tools I've written and use a lot in my day to day. Each project linked
below has extensive documentation and potentially a demo video to go with it.

# CLI

| Script       | Description                                                                |
|:-------------|:---------------------------------------------------------------------------|
| [bmsu]       | An rsync based backup script to gain control over your backups             |
| [invoice]    | Track and calculate how much money to bill your clients every month        |
| [lcurl]      | Run a specific curl command X number of times in a loop to detect downtime |
| [notes]      | A really simple way to manage your text based notes                        |
| [plutus]     | Quickly track and analyze your income and expenses (tracker)               |
| [wait-until] | Wait until a command of your choosing has run successfully                 |
| [webserver]  | A web server to echo back an HTTP request's headers and data               |

[bmsu]: https://github.com/nickjj/bmsu
[invoice]: https://github.com/nickjj/invoice
[lcurl]: https://github.com/nickjj/lcurl
[notes]: https://github.com/nickjj/notes
